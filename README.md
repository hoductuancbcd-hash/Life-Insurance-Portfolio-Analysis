# Life Insurance Portfolio & Premium Analysis

Đây là project cá nhân được xây dựng bằng **Microsoft Excel** nhằm phân tích cấu trúc của một danh mục bảo hiểm nhân thọ giả lập, tập trung vào **Portfolio Composition, Premium, Coverage và Customer Segmentation**.

Project sử dụng dataset gồm **10.000 hợp đồng bảo hiểm nhân thọ**, từ đó xây dựng quy trình:

**Raw Data → Analysis Data → Derived Metrics → Pivot Analysis → Interactive Dashboard**

## Mục tiêu Project

Project được thực hiện nhằm:

- Phân tích cơ cấu **Life Insurance Portfolio**
- So sánh Premium và Coverage giữa các **Policy Type**
- Phân tích Portfolio theo **Age Group, Income Band và Smoking Status**
- Xây dựng các chỉ số dẫn xuất hỗ trợ phân tích
- Sử dụng **PivotTables, PivotCharts và Slicers** để xây dựng Dashboard tương tác
- Chuyển dữ liệu thành các insight có thể hỗ trợ quá trình phân tích sâu hơn
